## Requirements:
    Install the Microsoft 2019 (!!!) Visual C++ Build Tools: https://visualstudio.microsoft.com/de/downloads/
        1) Run the installer.
        2) Select: Workloads ? Visual C++ build tools.
        3) Install options: select only the “Windows 10 SDK” (assuming the computer is Windows 10)
    
    Install poppler through conda: command: conda install -c conda-forge poppler (Anaconda or Miniconda (at least) must be installed)
